Overview of the Acceptability Rating Data of Japanese (ARDJ) Project, 2016-

This site describes a research project called "Acceptability Rating Data of Japanese" (ARDJ for short).

What Is ARDJ?

ARDJ is an acronym of Acceptability Rating Data (or Ratings) of Japanese. Why to build such data laboriously? To meet the need for making theoretical linguistics truly "evidence-based". In the current state of theoretical linguistics, evidence is largely anecdotal. The notion of evidence needs to be updated. ARDJ is a research project aiming at compiling a large database of acceptability from the view point of empirical science, thereby providing foundations for Evidence-based Linguistics in the same sense as Evidence-based Medicine.

What did ARDJ produce?

ARDJ released the following pieces of data for the time being:
  1. Responses for Survey 1 (grouped version with global S.ID) zipped xslx file
  2. Responses for Survey 1 (ungrouped version with local S.ID) zipped xslx file
  3. Responses for Survey 2 (Phases 1 and 2 unified) zipped tsv file and this data has its English documentation page.
  4. Collapsed responses for Survey 2 (Phases 1 and 2 unified) zipped tsv file.
  5. Reaction time data obtained using stimuli for Surveys 1 and 2 zipped .xlsx file and this data is going to have its English documentation page.
Note: X-refined1b-Y was replaced by X-refined1c-Y on 09/09, 2019. If these links don't work, visit the GitHub page.

Note: all pieces of data is free, but they are all password-protected. You need to contact site manager to get one. Why? Freely available data does not always mean freely distributable data. Freedom needs to be controlled and regulated. I don't like anarchy.
